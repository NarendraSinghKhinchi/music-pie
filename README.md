this is a music player app for couples.
they can play songs in sync.
if one partner uploads a song it automatically becomes available offline for the other partner.
they can like the songs.
they can create playlists.
they can chat while listening to music.
they can have moments from particular time stamps of the music.

once a member onboards he has option to invite only one partner 
each member has a unique id which he can share with other partner so that they can invite them to the pie-verse.
